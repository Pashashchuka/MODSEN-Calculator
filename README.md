# MODSEN-Calculator

#### Start ESlint

- Check eslint:

  `yarn run lint`

- Check eslint and autofix:

  `yarn run lint:fix`

#### Start

- Start project:

  `yarn run start`

#### Build

- Build project:

  `yarn run build`

#### Tests

- Start tests in command line:

  `yarn run cypress:run`

- Start tests in GUI

  `yarn run cypress`

#### Deploy

https://pashashchuka-modsen-calculator.netlify.app/
